# InCinema
InCinema это учебный проект, бекэнд базы фильмов, написанный на c#, 
с использованием Asp.net core

---

## Какие технологии используются?
* Dapper
* Automapper

---

## Какая используюется СУБД?
Microsoft SQL Server

---

## Какие фичи реализованы?
* Фильмы
  * Жанры
  

* Кино-персоны
  * Карьеры
